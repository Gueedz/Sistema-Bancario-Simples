# Sistema Bancário Simples

Meu segundo projeto em Java, desenvolvido com o objetivo de praticar conceitos fundamentais de Programação Orientada a Objetos (POO).

Funcionalidades: 
- Criar conta bancária
- Depositar valores
- Sacar valores
- Consultar saldo

Conceitos utilizados:
Classes e Objetos
Encapsulamento
Construtores
Métodos de negócio
Tratamento de exceções

Este projeto foi criado para reforçar os conhecimentos adquiridos durante meus estudos em Java, colocando em prática a modelagem de classes, regras de negócio e interação com o usuário via console.
